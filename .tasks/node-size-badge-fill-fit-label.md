# feat: показывать Fill/Fit в бейдже размера ноды

Рядом с числовым значением размера в бейдже ноды (который отображается при наведении/выделении) необходимо показывать текстовую метку Fill или Fit, если соответствующий режим задан в свойствах ноды (width/height = "fill_container" или "hug_contents").

## Шаги проверки
- Создать фрейм с auto-layout
- Добавить дочернюю ноду и задать ей width = fill_container
- Убедиться, что в бейдже размера рядом с шириной отображается метка "Fill"
- Задать дочерней ноде height = hug_contents
- Убедиться, что в бейдже размера рядом с высотой отображается метка "Fit"
- Убедиться, что при обычных числовых размерах метки Fill/Fit не отображаются
- Проверить корректность отображения бейджа при мультиселекте нод с разными режимами размера
